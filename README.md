# FIFA_SEARCHtoPlay

 ⚽ FIFA Players Manager (XAcademy Final Project)

Bienvenido al proyecto final de XAcademy: una aplicación web fullstack para gestionar jugadores del FIFA (2015–2023).  
💻 Backend con Node.js + MySQL · 🧩 Frontend en Angular · 🐳 Docker Ready

 📌 ¿Qué hace esta app?

 🔍 Buscar jugadores por nombre, club o posición
 📄 Exportar resultados a CSV
 📊 Ver detalles y habilidades con gráficos
 ✏️ Editar información del jugador
 🧑‍💻 Crear tu propio jugador
 🔐 Login obligatorio para acceder
 ⏳ Línea de tiempo de skills (próximamente)



 🚀 Tecnologías Usadas

| Parte      | Tech stack                        |
|------------|-----------------------------------|
| Frontend   | Angular, RxJS, Chart.js           |
| Backend    | Node.js, Express, Sequelize       |
| Auth       | JSON Web Tokens (JWT)             |
| Base de datos | MySQL                          |
| Docker     | Docker + Docker Compose           |
| Docs       | Postman / Swagger (a elección)    |
